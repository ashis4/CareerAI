Model Architecture
Keras Sequential Neural Network
Trained on skill-to-career dataset · Softmax output over 16 career classes
1
Embedding Layer
Converts token IDs to dense vectors — learns semantic meaning of each skill word during training.
2
GlobalAveragePooling1D
Aggregates all skill token embeddings into a single fixed-size feature vector for the input.
3
Dense Layer (ReLU)
Fully connected hidden layer that extracts high-level patterns and skill-role relationships.
4
Dense Layer (Softmax)
Output layer producing a probability distribution across all 16 career class labels.
